This log tracks daily accumulation of serosanguineous fluid from a JP Drain following a parotidectomy to remove tissue from the parotid, submandibular, and lymph node for biopsy. The drain was cleaned with alcohol each time, and the wound kept uncovered and clean. Daily output of less than 25mL is considered appropriate for drain removal, and it seems logical that the timing of this event would vary based on the procedure. This log aims to help pinpoint this event for a single case. This task is proving to be complicated, as the current rate is trending upward.
